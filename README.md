## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/juniorknx/netflixclone.git
```

Entre no diretório do projeto

```bash
  cd myflix
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
